About the project

Project GitHub Repository

This website allows an user to upload image or video. The detector detect two classes of objects - motorcycles and helmets. The website uses pre-trained Yolov8x model to inference results to the user. The model has been trained and validated on Open Images Dataset V7. The author used various tools to achieve project aims including : Yolov8, Bootstrap 5, Flask and Open Images Dataset V7. Please click links at the top right corner if interested in any of tools used by the author. This website is a part of Marek’s Checinski (Student Ref: 10804812) master’s dissertation – MSc AI student at Plymouth University.

The dataset for the project has been ethically retrieved from Open Images Dataset V7 API. The author used various Python code sources to download and pre-format data into YOLO format. The dataset contains 13.499 training and 417 validation images in various sizes for both clasess altogether. The images are downsized to 640x640 pixels at the input.

The best performing YOLOv8l model has been trained for 100 epochs with batch size equals to 16 images. The model presents performance:
mAP@50=0.792
mAP@50-90=0.562
Precision=0.735
Recall=0.772
